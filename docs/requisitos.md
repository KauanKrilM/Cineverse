# Requisitos Funcionais

## RF01
O sistema deve permitir navegação completa por teclado.

## RF02
O sistema deve possuir modo de alto contraste.

## RF03
O sistema deve permitir alteração do tamanho da fonte.

## RF04
O sistema deve integrar VLibras.

## RF05
O sistema deve utilizar animações com GSAP.

## RF06
O sistema deve conter elementos 3D com Three.js.

---

# Requisitos Não Funcionais

## RNF01
O sistema deve ser responsivo.

## RNF02
O sistema deve possuir interface acessível.

## RNF03
O sistema deve funcionar em navegadores modernos.
