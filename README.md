# CineVerse

Interactive and accessible movie experience website using Three.js, GSAP and VLibras.

CineVerse é um site interativo focado em experiência cinematográfica acessível, utilizando recursos modernos como animações com GSAP, elementos 3D com Three.js e ferramentas de acessibilidade como VLibras, navegação por teclado, alto contraste e controle de fonte.

## Tecnologias

- HTML
- CSS
- JavaScript
- Three.js
- GSAP
- VLibras

## Recursos de Acessibilidade

- Navegação por teclado
- Alto contraste
- Alteração de fonte
- Compatibilidade com leitores de tela
- Tradução em Libras

## Objetivo

Criar uma experiência cinematográfica imersiva e acessível utilizando tecnologias modernas da web.
